OpenClaude is a fork of Claude Code that replaces the Anthropic-only backend with an OpenAI-compatible provider shim. Every tool, slash command, sub-agent, and memory feature works exactly as before — backed by whatever model you choose.

## Get started in three steps

<Steps>
  <Step title="Install OpenClaude">
    ```bash
    npm install -g @gitlawb/openclaude
    ```
    Requires Node.js 20 or newer.
  </Step>
  <Step title="Set your provider">
    Pick any OpenAI-compatible provider and export three variables:
    ```bash
    export CLAUDE_CODE_USE_OPENAI=1
    export OPENAI_API_KEY=sk-your-key-here
    export OPENAI_MODEL=gpt-4o
    ```
    For local inference with Ollama, no API key is needed — just set `OPENAI_BASE_URL=http://localhost:11434/v1`.
  </Step>
  <Step title="Run OpenClaude">
    ```bash
    openclaude
    ```
    The CLI starts and waits for your prompt. Ask it to explain code, edit files, run commands, or review changes.
  </Step>
</Steps>
